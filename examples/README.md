# Examples

This folder contains illustrative output from the TFL Validator run against the included sample data.

## TFL_Validation_Report_EXAMPLE.xlsx

A sample validation report generated by running `python demo.py` against the synthetic oncology study dataset (`sample_data/`).

### What's inside:

| Sheet | Description |
|-------|-------------|
| **SUMMARY** | KPI dashboard — total TFLs, pass/fail counts, match rates |
| **T-01 … L-05** | Per-TFL detail — row-by-row comparison of TFL values vs recalculated values, color-coded PASS/FAIL |
| **AUDIT LOG** | Full calculation audit trail for regulatory traceability |
| **CONFIG** | Study configuration snapshot for reproducibility |

### Note

> This report was generated from **synthetic sample data only** and does not represent a real clinical trial.
> Run `python demo.py` to generate a fresh report on your own machine.
